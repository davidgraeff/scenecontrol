Konfigurationshinweise

Der SceneServer™ organisiert seine Funktionalität in Form von Plugins. Ein PLugin stellt Kontakt zu einer bestimmten Hardware her (z.B. HiFi-Receiver, Projektor, Licht) oder bietet bestimmte Funktionalität an (z.B. Ereignisauslösung durch Zeitpunkt, Steuerung des Medienplayers).

Um die Möglichkeit zu bieten, mehrere gleichartige Geräte oder Software anzusteuern, können von einem Plugin mehrere Kopien (Instanzen) gestartet werden. Für ein Plugin können daher mehrere Konfigurationen eingerichtet werden. Der SceneServer™ wertet die Konfigurationen aus und startet die benötigte Anzahl von Plugins.

Im linken Bereich dieser Seite sind die installierten Plugins aufgelistet. Sobald ein Eintrag ausgewählt wurde, wird die Hauptansicht, wo sich aktuell dieser Text befindet, mit den eingerichteten Konfiguration gefüllt. Konfigurationen können durch das Minussymbol im oberen, linken Bereich der jeweiligen Konfiguration gelöscht werden. Die zugehörige Plugininstanz wird dadurch vom SceneServer™ beendet. Im unteren, rechten Bereich kann mit der Schaltfläche "Konfiguration erstellen" eine neue Konfiguration erstellt werden. Sobald diese gespeichert wurde, wird der SceneServer™ eine entsprechene Plugininstanz starten.
